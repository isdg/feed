# ClickHouse Cloud on Google Cloud Platform (GCP) is Generally Available

ClickHouse Cloud is now Generally Available on Google Cloud Platform (GCP) in three geographies - United States (Iowa), Europe (Netherlands) and Asia (Singapore).
