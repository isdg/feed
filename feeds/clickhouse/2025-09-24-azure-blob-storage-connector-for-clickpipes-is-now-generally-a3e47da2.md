# Azure Blob Storage connector for ClickPipes is now Generally Available

We’re excited to announce the general availability of the Azure Blob Storage connector for ClickPipes. Natively integrated with ClickHouse Cloud, this connector enables blazing-fast load from Azure Blob Storage with just a few clicks.
