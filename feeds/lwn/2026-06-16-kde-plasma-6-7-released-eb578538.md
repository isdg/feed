# KDE Plasma 6.7 released

[Version
6.7](https://kde.org/announcements/plasma/6/6.7.0/) of KDE's Plasma desktop has been released. Notable changes in this release include per-screen virtual desktops, faster desktop switching, introduction of the [Union
theming system](https://quantumproductions.info/articles/2026-05/union-spring-2026-update) as a tech preview, as well as [many other improvements](https://kde.org/announcements/plasma/6/6.7.0/#but-thats-not-all) and bug fixes. The release is dedicated to Eric Laffoon, a longtime KDE supporter, who passed away in May.

See the [KDE
wiki](https://community.kde.org/Plasma/Plasma_6#Plasma_6.7) for a full list of new features, and the [Changelog](https://kde.org/announcements/changelogs/plasma/6/6.6.5-6.7.0/) for a list of all commits in this release.
