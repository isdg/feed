# GIMP 0.54.1 in a Flatpak

The GIMP project [reports](https://floss.social/@GIMP/116782353256793213) that GNOME contributor "balooii" has worked to package GIMP 0.54.1—released in 1996—as a Flatpak that will build and run on modern 64-bit Linux systems. This is a [Motif](https://en.wikipedia.org/wiki/Motif_(software))-based version, and the same version that was [used
by Larry Ewing](https://web.archive.org/web/19990208225150/http://www.isc.tamu.edu/%7Elewing/linux/notes.html) to create Tux.

While not likely to be useful for serious graphics work today, it should be interesting for users who would like to see what a 30-year-old version of GIMP was capable of.
