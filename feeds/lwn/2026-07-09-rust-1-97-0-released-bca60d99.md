# Rust 1.97.0 released

[Version
1.97.0](https://blog.rust-lang.org/2026/07/09/Rust-1.97.0/) of the Rust programming language has been released. Changes include using a new symbol-mangling scheme by default, support for denying warnings in Cargo, and an end to the practice of hiding the linker's output after a successful build.
