# The current state of MiniRust

A few weeks ago, many Rust folks met in Utrecht for RustWeek and we all had a great time. As part of that, I also gave a talk titled “MiniRust: A core language for specifying Rust” about the current state of MiniRust. This was my first time giving a talk in a (fully packed) movie theater; unfortunately, my special effects budget cannot keep up with the shows that would usually be presented there. But nevertheless, if you would like to learn more about my vision for how we should specify the gnarly details of unsafe Rust, [please go watch my talk](https://www.youtube.com/watch?v=yoeuW_dSe0o). :)

Thanks to everyone who was there for being a great audience, and thanks to the organizers for an amazing week and high-quality recordings!
