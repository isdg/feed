# A digestion of unit distance constructions

Suppose that one has a set ![{P}](https://s0.wp.com/latex.php?latex=%7BP%7D&bg=ffffff&fg=000000&s=0&c=20201002) of ![{n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0&c=20201002) points in the plane, which we will think of as the complex plane ![{{\bf C}}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cbf+C%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002). Let ![{d_1(P)}](https://s0.wp.com/latex.php?latex=%7Bd_1%28P%29%7D&bg=ffffff&fg=000000&s=0&c=20201002) denote the number of unit distances determined by these points, i.e., pairs of points ![{p,q \in P}](https://s0.wp.com/latex.php?latex=%7Bp%2Cq+%5Cin+P%7D&bg=ffffff&fg=000000&s=0&c=20201002) whose displacement ![{w = q-p}](https://s0.wp.com/latex.php?latex=%7Bw+%3D+q-p%7D&bg=ffffff&fg=000000&s=0&c=20201002) obeys the equation

![\displaystyle w \overline{w} = 1. \ \ \ \ \ (1)](https://s0.wp.com/latex.php?latex=%5Cdisplaystyle+w+%5Coverline%7Bw%7D+%3D+1.+%5C+%5C+%5C+%5C+%5C+%281%29&bg=ffffff&fg=000000&s=0&c=20201002)

(It makes little difference for the asymptotics, but we will count the pair ![{(q,p)}](https://s0.wp.com/latex.php?latex=%7B%28q%2Cp%29%7D&bg=ffffff&fg=000000&s=0&c=20201002) separately from ![{(p,q)}](https://s0.wp.com/latex.php?latex=%7B%28p%2Cq%29%7D&bg=ffffff&fg=000000&s=0&c=20201002) here.)

[The Erdös unit distance problem](https://www.erdosproblems.com/forum/thread/90) asks, for a given large number ![{n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0&c=20201002), what is the largest possible value of ![{d_1(P)}](https://s0.wp.com/latex.php?latex=%7Bd_1%28P%29%7D&bg=ffffff&fg=000000&s=0&c=20201002) amongst all sets ![{P}](https://s0.wp.com/latex.php?latex=%7BP%7D&bg=ffffff&fg=000000&s=0&c=20201002) of cardinality ![{n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0&c=20201002)?

For instance, if one takes ![{P}](https://s0.wp.com/latex.php?latex=%7BP%7D&bg=ffffff&fg=000000&s=0&c=20201002) to be ![{n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0&c=20201002) equally spaced collinear points with unit spacing, one can obtain a linear construction with ![{d_1(P) = 2n-2}](https://s0.wp.com/latex.php?latex=%7Bd_1%28P%29+%3D+2n-2%7D&bg=ffffff&fg=000000&s=0&c=20201002). Erdös observed that one can improve this construction asymptotically:

> **Theorem 1 (Erdös construction)** There exists point sets ![{P}](https://s0.wp.com/latex.php?latex=%7BP%7D&bg=ffffff&fg=000000&s=0&c=20201002) of arbitrarily large cardinality ![{n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0&c=20201002) such that ![{d_1(P) \gg n^{1 +c / \log\log n}}](https://s0.wp.com/latex.php?latex=%7Bd_1%28P%29+%5Cgg+n%5E%7B1+%2Bc+%2F+%5Clog%5Clog+n%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) for some absolute constant ![{c > 0}](https://s0.wp.com/latex.php?latex=%7Bc+%3E+0%7D&bg=ffffff&fg=000000&s=0&c=20201002).

In fact, in the construction one could take ![{c}](https://s0.wp.com/latex.php?latex=%7Bc%7D&bg=ffffff&fg=000000&s=0&c=20201002) arbitrarily close to ![{\log 2}](https://s0.wp.com/latex.php?latex=%7B%5Clog+2%7D&bg=ffffff&fg=000000&s=0&c=20201002). Erdös famously asked whether ![{d_1(P)}](https://s0.wp.com/latex.php?latex=%7Bd_1%28P%29%7D&bg=ffffff&fg=000000&s=0&c=20201002) had to be bounded above by ![{n^{1+o(1)}}](https://s0.wp.com/latex.php?latex=%7Bn%5E%7B1%2Bo%281%29%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002); and for decades there was significant effort expended on upper bounding ![{d_1(P)}](https://s0.wp.com/latex.php?latex=%7Bd_1%28P%29%7D&bg=ffffff&fg=000000&s=0&c=20201002), with the best known upper bound being ![{d_1(P) \ll n^{4/3}}](https://s0.wp.com/latex.php?latex=%7Bd_1%28P%29+%5Cll+n%5E%7B4%2F3%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002), established by [by Spencer, Szemerédi, and Trotter](https://zbmath.org/0561.52008) in 1984. We will note here that it seems extremely difficult to improve this upper bound. One reason for this is that if one replaces the equation [(1)](#w-displace) with the superficially similar equation

![\displaystyle \mathrm{Im} w = (\mathrm{Re} w)^2 \ \ \ \ \ (2)](https://s0.wp.com/latex.php?latex=%5Cdisplaystyle+%5Cmathrm%7BIm%7D+w+%3D+%28%5Cmathrm%7BRe%7D+w%29%5E2+%5C+%5C+%5C+%5C+%5C+%282%29&bg=ffffff&fg=000000&s=0&c=20201002)

(i.e., replace the unit circle ![{(\mathrm{Re} w)^2 + (\mathrm{Im} w)^2 = 1}](https://s0.wp.com/latex.php?latex=%7B%28%5Cmathrm%7BRe%7D+w%29%5E2+%2B+%28%5Cmathrm%7BIm%7D+w%29%5E2+%3D+1%7D&bg=ffffff&fg=000000&s=0&c=20201002) by a standard parabola), then the ![{n^{4/3}}](https://s0.wp.com/latex.php?latex=%7Bn%5E%7B4%2F3%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) bound is best possible, as can be seen by taking ![{P}](https://s0.wp.com/latex.php?latex=%7BP%7D&bg=ffffff&fg=000000&s=0&c=20201002) to be a rectangle in the Gaussian integers of width ![{\asymp n^{1/3}}](https://s0.wp.com/latex.php?latex=%7B%5Casymp+n%5E%7B1%2F3%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) and height ![{\asymp n^{2/3}}](https://s0.wp.com/latex.php?latex=%7B%5Casymp+n%5E%7B2%2F3%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002). Hence any improvement of the ![{n^{4/3}}](https://s0.wp.com/latex.php?latex=%7Bn%5E%7B4%2F3%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) bound would have to exploit some special property of the unit circle that is not shared by the parabola.

It came as some surprise recently when a team from OpenAI [resolved the question of Erdös](https://cdn.openai.com/pdf/74c24085-19b0-4534-9c90-465b8e29ad73/unit-distance-proof.pdf):

> **Theorem 2 (OpenAI construction)** There exists point sets ![{P}](https://s0.wp.com/latex.php?latex=%7BP%7D&bg=ffffff&fg=000000&s=0&c=20201002) of arbitrarily large cardinality ![{n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0&c=20201002) such that ![{d_1(P) \gg n^{1 + c}}](https://s0.wp.com/latex.php?latex=%7Bd_1%28P%29+%5Cgg+n%5E%7B1+%2B+c%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) for some absolute constant ![{c > 0}](https://s0.wp.com/latex.php?latex=%7Bc+%3E+0%7D&bg=ffffff&fg=000000&s=0&c=20201002).

The optimal value of ![{c}](https://s0.wp.com/latex.php?latex=%7Bc%7D&bg=ffffff&fg=000000&s=0&c=20201002) is still unknown, but the best upper and lower bounds on ![{c}](https://s0.wp.com/latex.php?latex=%7Bc%7D&bg=ffffff&fg=000000&s=0&c=20201002) are tracked at [this page](https://teorth.github.io/optimizationproblems/constants/84a.html); currently we know that ![{0.03583\dots \leq c \leq 1/3}](https://s0.wp.com/latex.php?latex=%7B0.03583%5Cdots+%5Cleq+c+%5Cleq+1%2F3%7D&bg=ffffff&fg=000000&s=0&c=20201002).

The construction in Theorem [2](#openai) is a heavily modified version of that in Theorem [1](#erdos), and uses some non-trivial amount of algebraic number theory, in particular the device of Golod–Shafarevich towers of field extensions. However, it was [later observed](https://x.com/__alpoge__/status/2059298565093196012) using the Mythos AI that one could get a weaker bound with less algebraic number theory, which after optimizing parameters yields the following intermediate result between Theorem [1](#erdos) and Theorem [2](#openai):

> **Theorem 3 (Mythos construction)** There exists point sets ![{P}](https://s0.wp.com/latex.php?latex=%7BP%7D&bg=ffffff&fg=000000&s=0&c=20201002) of arbitrarily large cardinality ![{n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0&c=20201002) such that ![{d_1(P) \gg n^{1 + c / \log\log\log n}}](https://s0.wp.com/latex.php?latex=%7Bd_1%28P%29+%5Cgg+n%5E%7B1+%2B+c+%2F+%5Clog%5Clog%5Clog+n%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) for some absolute constant ![{c > 0}](https://s0.wp.com/latex.php?latex=%7Bc+%3E+0%7D&bg=ffffff&fg=000000&s=0&c=20201002).

Furthermore, by inserting Golod–Shafarevich towers back into the Mythos construction, one can recover the full strength of Theorem [2](#openai).

These results already have a number of expositions; see for instance [this article of Alon et al.](https://arxiv.org/abs/2605.20695), or [this blog post of Bloom](https://www.erdosproblems.com/forum/thread/blog:6). As an exercise for myself, I recently spent some time trying to “digest” these constructions and place them on a common footing, with an emphasis on trying to find the minimal route to either heuristically or rigorously recovering these results relying on as little algebraic number theory as possible. The post here is a writeup of this exercise. (Disclosure: AI tools were useful for providing initial summaries of these arguments, as well as on explaining various fundamentals of algebraic number theory to me.)

The first (trivial) observation is that one can use rescaling to replace the unit distance by any other fixed distance. In particular, for any positive real ![{m}](https://s0.wp.com/latex.php?latex=%7Bm%7D&bg=ffffff&fg=000000&s=0&c=20201002), if we let ![{d_m(P)}](https://s0.wp.com/latex.php?latex=%7Bd_m%28P%29%7D&bg=ffffff&fg=000000&s=0&c=20201002) denote the number of pairs ![{p,q}](https://s0.wp.com/latex.php?latex=%7Bp%2Cq%7D&bg=ffffff&fg=000000&s=0&c=20201002) whose displacement ![{w = q-p}](https://s0.wp.com/latex.php?latex=%7Bw+%3D+q-p%7D&bg=ffffff&fg=000000&s=0&c=20201002) obeys the equation

![\displaystyle w \overline{w} = m, \ \ \ \ \ (3)](https://s0.wp.com/latex.php?latex=%5Cdisplaystyle+w+%5Coverline%7Bw%7D+%3D+m%2C+%5C+%5C+%5C+%5C+%5C+%283%29&bg=ffffff&fg=000000&s=0&c=20201002)

then it is clear that any construction of a point set ![{P}](https://s0.wp.com/latex.php?latex=%7BP%7D&bg=ffffff&fg=000000&s=0&c=20201002) with a given value of ![{d_m(P)}](https://s0.wp.com/latex.php?latex=%7Bd_m%28P%29%7D&bg=ffffff&fg=000000&s=0&c=20201002) can be rescaled to another point set ![{m^{-1/2} \cdot P}](https://s0.wp.com/latex.php?latex=%7Bm%5E%7B-1%2F2%7D+%5Ccdot+P%7D&bg=ffffff&fg=000000&s=0&c=20201002) of the same cardinality with the corresponding value of ![{d_1(P)}](https://s0.wp.com/latex.php?latex=%7Bd_1%28P%29%7D&bg=ffffff&fg=000000&s=0&c=20201002). It turns out to be convenient to work with values of ![{m}](https://s0.wp.com/latex.php?latex=%7Bm%7D&bg=ffffff&fg=000000&s=0&c=20201002) that are asymptotically large, for instance the product of several large primes.

All the constructions of good point sets ![{P}](https://s0.wp.com/latex.php?latex=%7BP%7D&bg=ffffff&fg=000000&s=0&c=20201002) basically involve taking all the elements of a certain ring ![{B}](https://s0.wp.com/latex.php?latex=%7BB%7D&bg=ffffff&fg=000000&s=0&c=20201002) of algebraic integers up to some height. In the original construction of Erdös, ![{B}](https://s0.wp.com/latex.php?latex=%7BB%7D&bg=ffffff&fg=000000&s=0&c=20201002) was chosen to be the ring of Gaussian integers ![{{\bf Z}[i]}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cbf+Z%7D%5Bi%5D%7D&bg=ffffff&fg=000000&s=0&c=20201002), but in fact any ring of integers in a non-trivial bounded degree field extension of ![{{\bf Q}}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cbf+Q%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) would suffice to recover Theorem [1](#erdos) (though always with the constant ![{c}](https://s0.wp.com/latex.php?latex=%7Bc%7D&bg=ffffff&fg=000000&s=0&c=20201002) not exceeding ![{\log 2}](https://s0.wp.com/latex.php?latex=%7B%5Clog+2%7D&bg=ffffff&fg=000000&s=0&c=20201002)). To go beyond this, one has to start considering number fields of unbounded degree. As it turns out, the field extensions arising from Golod–Shafarevich towers are the most efficient for this purpose, and lead to Theorem [2](#openai); but one can work with the more elementary construction of number fields generated by many square roots of medium-sized primes, and this suffices for the intermediate result in Theorem [3](#mythos).

The numerology can be explained as follows. Take ![{B}](https://s0.wp.com/latex.php?latex=%7BB%7D&bg=ffffff&fg=000000&s=0&c=20201002) to be a ring of integers in some number field of degree ![{D}](https://s0.wp.com/latex.php?latex=%7BD%7D&bg=ffffff&fg=000000&s=0&c=20201002), and suppose for sake of argument that ![{m}](https://s0.wp.com/latex.php?latex=%7Bm%7D&bg=ffffff&fg=000000&s=0&c=20201002) is the product of ![{t}](https://s0.wp.com/latex.php?latex=%7Bt%7D&bg=ffffff&fg=000000&s=0&c=20201002) (rational) primes ![{p_1,\dots,p_t}](https://s0.wp.com/latex.php?latex=%7Bp_1%2C%5Cdots%2Cp_t%7D&bg=ffffff&fg=000000&s=0&c=20201002), which for simplicity we will assume to all have comparable magnitude, thus ![{p_i \asymp T}](https://s0.wp.com/latex.php?latex=%7Bp_i+%5Casymp+T%7D&bg=ffffff&fg=000000&s=0&c=20201002) for some ![{T}](https://s0.wp.com/latex.php?latex=%7BT%7D&bg=ffffff&fg=000000&s=0&c=20201002) and all ![{i=1,\dots,t}](https://s0.wp.com/latex.php?latex=%7Bi%3D1%2C%5Cdots%2Ct%7D&bg=ffffff&fg=000000&s=0&c=20201002). Thus, ![{m}](https://s0.wp.com/latex.php?latex=%7Bm%7D&bg=ffffff&fg=000000&s=0&c=20201002) is roughly of the size of ![{T^t}](https://s0.wp.com/latex.php?latex=%7BT%5Et%7D&bg=ffffff&fg=000000&s=0&c=20201002). In practice one wants to impose some additional “splitting” conditions on these primes ![{p_1,\dots,p_t}](https://s0.wp.com/latex.php?latex=%7Bp_1%2C%5Cdots%2Cp_t%7D&bg=ffffff&fg=000000&s=0&c=20201002), but the prime number theorem, as well as variants such as the Chebotarev density theorem, suggest that we should be able to keep ![{T}](https://s0.wp.com/latex.php?latex=%7BT%7D&bg=ffffff&fg=000000&s=0&c=20201002) reasonably close to ![{t}](https://s0.wp.com/latex.php?latex=%7Bt%7D&bg=ffffff&fg=000000&s=0&c=20201002) in size; for instance, if we select primes greedily then we can have ![{T \asymp t \log t}](https://s0.wp.com/latex.php?latex=%7BT+%5Casymp+t+%5Clog+t%7D&bg=ffffff&fg=000000&s=0&c=20201002). In particular we expect to have ![{\log T \asymp \log t}](https://s0.wp.com/latex.php?latex=%7B%5Clog+T+%5Casymp+%5Clog+t%7D&bg=ffffff&fg=000000&s=0&c=20201002) in practice.

By construction, ![{m}](https://s0.wp.com/latex.php?latex=%7Bm%7D&bg=ffffff&fg=000000&s=0&c=20201002) splits into the product of ![{t}](https://s0.wp.com/latex.php?latex=%7Bt%7D&bg=ffffff&fg=000000&s=0&c=20201002) rational primes. Moving up to the degree ![{D}](https://s0.wp.com/latex.php?latex=%7BD%7D&bg=ffffff&fg=000000&s=0&c=20201002) extension, one can optimistically hope that ![{m}](https://s0.wp.com/latex.php?latex=%7Bm%7D&bg=ffffff&fg=000000&s=0&c=20201002) splits further into the product of ![{Dt}](https://s0.wp.com/latex.php?latex=%7BDt%7D&bg=ffffff&fg=000000&s=0&c=20201002) primes in ![{B}](https://s0.wp.com/latex.php?latex=%7BB%7D&bg=ffffff&fg=000000&s=0&c=20201002). Using conjugation symmetry, these primes might split into ![{Dt/2}](https://s0.wp.com/latex.php?latex=%7BDt%2F2%7D&bg=ffffff&fg=000000&s=0&c=20201002) conjugate pairs ![{\rho, \overline{\rho}}](https://s0.wp.com/latex.php?latex=%7B%5Crho%2C+%5Coverline%7B%5Crho%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002). By selecting one element from each pair and multiplying, this generates ![{2^{Dt/2}}](https://s0.wp.com/latex.php?latex=%7B2%5E%7BDt%2F2%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) solutions to [(3)](#w-displace-mu) in ![{B}](https://s0.wp.com/latex.php?latex=%7BB%7D&bg=ffffff&fg=000000&s=0&c=20201002). These solutions ![{w}](https://s0.wp.com/latex.php?latex=%7Bw%7D&bg=ffffff&fg=000000&s=0&c=20201002) will of course have complex magnitude ![{m^{1/2}}](https://s0.wp.com/latex.php?latex=%7Bm%5E%7B1%2F2%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002); one can optimistically hope that they in fact have “height” ![{O(m^{1/2})}](https://s0.wp.com/latex.php?latex=%7BO%28m%5E%7B1%2F2%7D%29%7D&bg=ffffff&fg=000000&s=0&c=20201002) in some sense.

To take advantage of this, take ![{P}](https://s0.wp.com/latex.php?latex=%7BP%7D&bg=ffffff&fg=000000&s=0&c=20201002) to be the set of points in ![{B}](https://s0.wp.com/latex.php?latex=%7BB%7D&bg=ffffff&fg=000000&s=0&c=20201002) of height ![{O(m^{1/2})}](https://s0.wp.com/latex.php?latex=%7BO%28m%5E%7B1%2F2%7D%29%7D&bg=ffffff&fg=000000&s=0&c=20201002). As ![{B}](https://s0.wp.com/latex.php?latex=%7BB%7D&bg=ffffff&fg=000000&s=0&c=20201002) has rank ![{D}](https://s0.wp.com/latex.php?latex=%7BD%7D&bg=ffffff&fg=000000&s=0&c=20201002), we therefore expect the size ![{n = |P|}](https://s0.wp.com/latex.php?latex=%7Bn+%3D+%7CP%7C%7D&bg=ffffff&fg=000000&s=0&c=20201002) of this set to be roughly

![\displaystyle n \approx (m^{1/2})^D \approx T^{D t / 2}. \ \ \ \ \ (4)](https://s0.wp.com/latex.php?latex=%5Cdisplaystyle+n+%5Capprox+%28m%5E%7B1%2F2%7D%29%5ED+%5Capprox+T%5E%7BD+t+%2F+2%7D.+%5C+%5C+%5C+%5C+%5C+%284%29&bg=ffffff&fg=000000&s=0&c=20201002)

(For this heuristic discussion I will be deliberately vague about what the symbol ![{\approx}](https://s0.wp.com/latex.php?latex=%7B%5Capprox%7D&bg=ffffff&fg=000000&s=0&c=20201002) means.) Meanwhile, using our solutions to [(3)](#w-displace-mu), we expect to have

![\displaystyle d_m(P) \gtrapprox n 2^{Dt/2} \approx n^{1 + \frac{\log 2}{\log T}} \approx n^{1 + \frac{\log 2}{\log t}}. \ \ \ \ \ (5)](https://s0.wp.com/latex.php?latex=%5Cdisplaystyle+d_m%28P%29+%5Cgtrapprox+n+2%5E%7BDt%2F2%7D+%5Capprox+n%5E%7B1+%2B+%5Cfrac%7B%5Clog+2%7D%7B%5Clog+T%7D%7D+%5Capprox+n%5E%7B1+%2B+%5Cfrac%7B%5Clog+2%7D%7B%5Clog+t%7D%7D.+%5C+%5C+%5C+%5C+%5C+%285%29&bg=ffffff&fg=000000&s=0&c=20201002)

But this can be clarified by the heuristic [(4)](#nlogn). Taking logarithms, we expect to have

![\displaystyle \log n \asymp D t \log T \asymp D t \log t. \ \ \ \ \ (6)](https://s0.wp.com/latex.php?latex=%5Cdisplaystyle+%5Clog+n+%5Casymp+D+t+%5Clog+T+%5Casymp+D+t+%5Clog+t.+%5C+%5C+%5C+%5C+%5C+%286%29&bg=ffffff&fg=000000&s=0&c=20201002)

In the regime where the degree ![{D}](https://s0.wp.com/latex.php?latex=%7BD%7D&bg=ffffff&fg=000000&s=0&c=20201002) of the number field is held fixed, we thus expect ![{t}](https://s0.wp.com/latex.php?latex=%7Bt%7D&bg=ffffff&fg=000000&s=0&c=20201002) to exhibit logarithmic type growth in ![{n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0&c=20201002), and on inserting this back into [(5)](#dmu) we (heuristically) recover Theorem [1](#erdos) (with the natural constant ![{c = \log 2}](https://s0.wp.com/latex.php?latex=%7Bc+%3D+%5Clog+2%7D&bg=ffffff&fg=000000&s=0&c=20201002)). In fact it is not hard to turn the above heuristics into a rigorous argument, by setting ![{B}](https://s0.wp.com/latex.php?latex=%7BB%7D&bg=ffffff&fg=000000&s=0&c=20201002) equal the Gaussian integers ![{{\bf Z}[i]}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cbf+Z%7D%5Bi%5D%7D&bg=ffffff&fg=000000&s=0&c=20201002) and selecting all the primes ![{p_1,\dots,p_t}](https://s0.wp.com/latex.php?latex=%7Bp_1%2C%5Cdots%2Cp_t%7D&bg=ffffff&fg=000000&s=0&c=20201002) to be ![{1 \hbox{ mod } 4}](https://s0.wp.com/latex.php?latex=%7B1+%5Chbox%7B+mod+%7D+4%7D&bg=ffffff&fg=000000&s=0&c=20201002), so that they split completely in ![{{\bf Z}[i]}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cbf+Z%7D%5Bi%5D%7D&bg=ffffff&fg=000000&s=0&c=20201002) by the [Fermat two-square theorem](https://en.wikipedia.org/wiki/Fermat%27s_theorem_on_sums_of_two_squares).

But if one can permit the degree ![{D}](https://s0.wp.com/latex.php?latex=%7BD%7D&bg=ffffff&fg=000000&s=0&c=20201002) to grow in the construction, and in particular be superpolynomial in ![{t}](https://s0.wp.com/latex.php?latex=%7Bt%7D&bg=ffffff&fg=000000&s=0&c=20201002), then the above heuristics suggest that we can start improving upon Theorem [1](#erdos) , and even get all the way to Theorem [2](#openai) if we can make the degree go to infinity while keeping the number the number of primes ![{t}](https://s0.wp.com/latex.php?latex=%7Bt%7D&bg=ffffff&fg=000000&s=0&c=20201002).

If one naively tries this approach by forcing all the primes ![{p}](https://s0.wp.com/latex.php?latex=%7Bp%7D&bg=ffffff&fg=000000&s=0&c=20201002) to split completely in a very high degree number field, one runs into significant technical difficulties, not least of which is the need to obtain good error terms in the [Chebotarev density theorem](https://en.wikipedia.org/wiki/Chebotarev_density_theorem), which touches upon such difficult questions as the Generalized Riemann Hypothesis and the existence of Siegel zeroes. From an algebraic number theory perspective, this is related to the breakdown of unique factorization in such number fields, as measured by the class group. The size of this group is in turn controlled by the discriminant of the field, as per the [fundamental theorem of Minkowski](https://en.wikipedia.org/wiki/Minkowski%27s_bound) in this subject.

But one can hope that the arguments are robust enough to tolerate a little bit of breakdown in unique factorization, so long as the class group is not too large. The most natural way to do this is to use all the standard machinery of algebraic number theory, such as the unique factorization of ideals. But there turns out to be a more elementary (though largely equivalent) approach, which is to weaken the target equation [(3)](#w-displace-mu) to a congruence equation

![\displaystyle w \overline{w} = 0 \hbox{ mod } mB. \ \ \ \ \ (7)](https://s0.wp.com/latex.php?latex=%5Cdisplaystyle+w+%5Coverline%7Bw%7D+%3D+0+%5Chbox%7B+mod+%7D+mB.+%5C+%5C+%5C+%5C+%5C+%287%29&bg=ffffff&fg=000000&s=0&c=20201002)

This condition does not pin down the value of ![{w \overline{w}}](https://s0.wp.com/latex.php?latex=%7Bw+%5Coverline%7Bw%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) completely, but so long as we can keep the height of ![{w}](https://s0.wp.com/latex.php?latex=%7Bw%7D&bg=ffffff&fg=000000&s=0&c=20201002) not too much larger than ![{m^{1/2}}](https://s0.wp.com/latex.php?latex=%7Bm%5E%7B1%2F2%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002), it does restrict ![{w \overline{w}}](https://s0.wp.com/latex.php?latex=%7Bw+%5Coverline%7Bw%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) to a sufficiently small set of possible values that a simple application of the pigeonhole principle can allow one to conclude.

In order for this strategy to work well, one needs to locate high-degree number fields ![{B}](https://s0.wp.com/latex.php?latex=%7BB%7D&bg=ffffff&fg=000000&s=0&c=20201002) of controlled discriminant for which it is relatively easy to at least partially split one’s rational primes ![{p_1,\dots,p_t}](https://s0.wp.com/latex.php?latex=%7Bp_1%2C%5Cdots%2Cp_t%7D&bg=ffffff&fg=000000&s=0&c=20201002) into ideals in this field ![{B}](https://s0.wp.com/latex.php?latex=%7BB%7D&bg=ffffff&fg=000000&s=0&c=20201002). It turns out that requiring the field to have a tower structure and admit [complex multiplication](https://en.wikipedia.org/wiki/Complex_multiplication) (which basically amounts to it including ![{i}](https://s0.wp.com/latex.php?latex=%7Bi%7D&bg=ffffff&fg=000000&s=0&c=20201002)) is already sufficient to get a satisfactory amount of splitting. To control discriminants, the most efficient choices are the Golod–Shafarevich towers, for which the (root) discriminant stays bounded; but a more naive choice of a tower of quadratic extensions also gives reasonable control on discriminants and is sufficient to establish Theorem [3](#mythos).

The three constructions thus sit on a continuum, with the key differences being the selection of the key parameters ![{t}](https://s0.wp.com/latex.php?latex=%7Bt%7D&bg=ffffff&fg=000000&s=0&c=20201002) (the number of primes multiplied together) and ![{D}](https://s0.wp.com/latex.php?latex=%7BD%7D&bg=ffffff&fg=000000&s=0&c=20201002) (the degree). The Erdös construction keeps the degree fixed and sends the number of primes to infinity. The OpenAI construction does the opposite, keeping the set of primes fixed but sending the degree to infinity. The Mythos construction is a compromise, in which the degree and the number of primes both go to infinity in a coupled fashion. In particular, one could easily imagine an alternate timeline of events in which the Mythos construction was the first to be discovered (by either humans or AI) after the Erdos construction as a reasonably natural modification of the latter, and then subsequently refined (again either by humans or AI) to the OpenAI construction once the significance of Golod–Shafarevich towers was realized.

In [this recent paper of Pohoata](https://arxiv.org/abs/2606.13619), the terms “horizontal amplification” and “vertical amplification” were proposed for the technique of constructing large configurations by increasing ![{t}](https://s0.wp.com/latex.php?latex=%7Bt%7D&bg=ffffff&fg=000000&s=0&c=20201002) and ![{D}](https://s0.wp.com/latex.php?latex=%7BD%7D&bg=ffffff&fg=000000&s=0&c=20201002), thus the Erdös construction becomes a paradigm for horizontal amplification while the OpenAI construction becomes a paradigm for vertical amplification (and the Mythos construction utilizes both types of amplification). See also [this paper of Bloom-Sawin-Schildkraut-Zhelezov](https://arxiv.org/abs/2605.28781) for another recent application of vertical amplification.

**— 1\. Some more details —**

Here we sketch how the quadratic extension approach can recover Theorem [3](#mythos).

As indicated above, we will work with a product ![{m = p_1 \dots p_t}](https://s0.wp.com/latex.php?latex=%7Bm+%3D+p_1+%5Cdots+p_t%7D&bg=ffffff&fg=000000&s=0&c=20201002) of ![{t}](https://s0.wp.com/latex.php?latex=%7Bt%7D&bg=ffffff&fg=000000&s=0&c=20201002) (rational) primes of size ![{\asymp T}](https://s0.wp.com/latex.php?latex=%7B%5Casymp+T%7D&bg=ffffff&fg=000000&s=0&c=20201002). Our only requirements of these primes, beyond their size, will be that they are distinct and equal to ![{1}](https://s0.wp.com/latex.php?latex=%7B1%7D&bg=ffffff&fg=000000&s=0&c=20201002) mod ![{4}](https://s0.wp.com/latex.php?latex=%7B4%7D&bg=ffffff&fg=000000&s=0&c=20201002), so that they split in the Gaussian integers. By the prime number theorem in arithmetic progressions, this allows us to take ![{T}](https://s0.wp.com/latex.php?latex=%7BT%7D&bg=ffffff&fg=000000&s=0&c=20201002) as small as ![{T \asymp t \log t}](https://s0.wp.com/latex.php?latex=%7BT+%5Casymp+t+%5Clog+t%7D&bg=ffffff&fg=000000&s=0&c=20201002), so in particular ![{\log T \asymp \log t}](https://s0.wp.com/latex.php?latex=%7B%5Clog+T+%5Casymp+%5Clog+t%7D&bg=ffffff&fg=000000&s=0&c=20201002).

To construct ![{B}](https://s0.wp.com/latex.php?latex=%7BB%7D&bg=ffffff&fg=000000&s=0&c=20201002), we start by taking ![{g}](https://s0.wp.com/latex.php?latex=%7Bg%7D&bg=ffffff&fg=000000&s=0&c=20201002) distinct medium-sized (rational) primes ![{q_1,\dots,q_g}](https://s0.wp.com/latex.php?latex=%7Bq_1%2C%5Cdots%2Cq_g%7D&bg=ffffff&fg=000000&s=0&c=20201002) of size ![{\asymp G}](https://s0.wp.com/latex.php?latex=%7B%5Casymp+G%7D&bg=ffffff&fg=000000&s=0&c=20201002) for some medium-sized parameter ![{g}](https://s0.wp.com/latex.php?latex=%7Bg%7D&bg=ffffff&fg=000000&s=0&c=20201002) (eventually, when we optimize parameters, we will take ![{g}](https://s0.wp.com/latex.php?latex=%7Bg%7D&bg=ffffff&fg=000000&s=0&c=20201002) to be a small multiple of ![{t/\log t}](https://s0.wp.com/latex.php?latex=%7Bt%2F%5Clog+t%7D&bg=ffffff&fg=000000&s=0&c=20201002)). We will not need any further properties of these primes, so by the prime number theorem we can take ![{G}](https://s0.wp.com/latex.php?latex=%7BG%7D&bg=ffffff&fg=000000&s=0&c=20201002) as small as ![{G \asymp g \log g}](https://s0.wp.com/latex.php?latex=%7BG+%5Casymp+g+%5Clog+g%7D&bg=ffffff&fg=000000&s=0&c=20201002), so in particular ![{\log G \asymp \log g}](https://s0.wp.com/latex.php?latex=%7B%5Clog+G+%5Casymp+%5Clog+g%7D&bg=ffffff&fg=000000&s=0&c=20201002). We will take ![{T}](https://s0.wp.com/latex.php?latex=%7BT%7D&bg=ffffff&fg=000000&s=0&c=20201002) to be larger than ![{G}](https://s0.wp.com/latex.php?latex=%7BG%7D&bg=ffffff&fg=000000&s=0&c=20201002), so that the primes ![{p_1,\dots,p_t}](https://s0.wp.com/latex.php?latex=%7Bp_1%2C%5Cdots%2Cp_t%7D&bg=ffffff&fg=000000&s=0&c=20201002) are distinct from the primes ![{q_1,\dots,q_g}](https://s0.wp.com/latex.php?latex=%7Bq_1%2C%5Cdots%2Cq_g%7D&bg=ffffff&fg=000000&s=0&c=20201002).

We will work in the number field ![\displaystyle {\bf Q}(i, \sqrt{q_1}, \dots, \sqrt{q_g})](https://s0.wp.com/latex.php?latex=%5Cdisplaystyle++%7B%5Cbf+Q%7D%28i%2C+%5Csqrt%7Bq_1%7D%2C+%5Cdots%2C+%5Csqrt%7Bq_g%7D%29&bg=ffffff&fg=000000&s=0&c=20201002)

generated by ![{i}](https://s0.wp.com/latex.php?latex=%7Bi%7D&bg=ffffff&fg=000000&s=0&c=20201002) and ![{g}](https://s0.wp.com/latex.php?latex=%7Bg%7D&bg=ffffff&fg=000000&s=0&c=20201002) real square roots ![{\sqrt{q_1},\dots,\sqrt{q_g}}](https://s0.wp.com/latex.php?latex=%7B%5Csqrt%7Bq_1%7D%2C%5Cdots%2C%5Csqrt%7Bq_g%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002). For instance, if ![{g = 1}](https://s0.wp.com/latex.php?latex=%7Bg+%3D+1%7D&bg=ffffff&fg=000000&s=0&c=20201002) and ![{q_1 = 3}](https://s0.wp.com/latex.php?latex=%7Bq_1+%3D+3%7D&bg=ffffff&fg=000000&s=0&c=20201002), a typical element in this field would take the form

![\displaystyle a + b i + c \sqrt{3} + d i \sqrt{3} \ \ \ \ \ (8)](https://s0.wp.com/latex.php?latex=%5Cdisplaystyle++a+%2B+b+i+%2B+c+%5Csqrt%7B3%7D+%2B+d+i+%5Csqrt%7B3%7D+%5C+%5C+%5C+%5C+%5C+%288%29&bg=ffffff&fg=000000&s=0&c=20201002)

In this particular example, the ring of integers would consist of those elements [(8)](#abcd) for which ![{a,b,c,d}](https://s0.wp.com/latex.php?latex=%7Ba%2Cb%2Cc%2Cd%7D&bg=ffffff&fg=000000&s=0&c=20201002) are rational integers. In general, the ring of integers can be slightly larger than this, but for our purposes we can just work with the “naive” ring of integers ![{B}](https://s0.wp.com/latex.php?latex=%7BB%7D&bg=ffffff&fg=000000&s=0&c=20201002) generated by ![{i}](https://s0.wp.com/latex.php?latex=%7Bi%7D&bg=ffffff&fg=000000&s=0&c=20201002) and ![{\sqrt{q_1},\dots,\sqrt{q_g}}](https://s0.wp.com/latex.php?latex=%7B%5Csqrt%7Bq_1%7D%2C%5Cdots%2C%5Csqrt%7Bq_g%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002). A typical element of this ring then looks like

![\displaystyle \sum_{I \subset \{0,\dots,g\}} a_I \prod_{i \in I} \sqrt{q_i}](https://s0.wp.com/latex.php?latex=%5Cdisplaystyle++%5Csum_%7BI+%5Csubset+%5C%7B0%2C%5Cdots%2Cg%5C%7D%7D+a_I+%5Cprod_%7Bi+%5Cin+I%7D+%5Csqrt%7Bq_i%7D&bg=ffffff&fg=000000&s=0&c=20201002) where ![{a_I}](https://s0.wp.com/latex.php?latex=%7Ba_I%7D&bg=ffffff&fg=000000&s=0&c=20201002) are rational integers and we adopt the convention that ![{\sqrt{q_0} = i}](https://s0.wp.com/latex.php?latex=%7B%5Csqrt%7Bq_0%7D+%3D+i%7D&bg=ffffff&fg=000000&s=0&c=20201002). Let us define the (naive) height of such a ring element to be ![{\max_I |a_I|}](https://s0.wp.com/latex.php?latex=%7B%5Cmax_I+%7Ca_I%7C%7D&bg=ffffff&fg=000000&s=0&c=20201002). Then the number of elements of ![{B}](https://s0.wp.com/latex.php?latex=%7BB%7D&bg=ffffff&fg=000000&s=0&c=20201002) of height at most ![{H}](https://s0.wp.com/latex.php?latex=%7BH%7D&bg=ffffff&fg=000000&s=0&c=20201002) is ![{\approx H^D}](https://s0.wp.com/latex.php?latex=%7B%5Capprox+H%5ED%7D&bg=ffffff&fg=000000&s=0&c=20201002) as long as ![{H}](https://s0.wp.com/latex.php?latex=%7BH%7D&bg=ffffff&fg=000000&s=0&c=20201002) is sufficiently large (again I will be vague about what ![{\approx}](https://s0.wp.com/latex.php?latex=%7B%5Capprox%7D&bg=ffffff&fg=000000&s=0&c=20201002) means here).

Our main goal is to find a large number of solutions ![{w}](https://s0.wp.com/latex.php?latex=%7Bw%7D&bg=ffffff&fg=000000&s=0&c=20201002) in ![{B}](https://s0.wp.com/latex.php?latex=%7BB%7D&bg=ffffff&fg=000000&s=0&c=20201002) to the congruence equation [(7)](#w-displace-cong). As per the usual [Minkowski embedding](https://en.wikipedia.org/wiki/Algebraic_number_theory#Real_and_complex_embeddings) based on the various ways to embed ![{B}](https://s0.wp.com/latex.php?latex=%7BB%7D&bg=ffffff&fg=000000&s=0&c=20201002) into ![{{\bf R}}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cbf+R%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) or ![{{\bf C}}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cbf+C%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002), it is convenient to think of ![{B}](https://s0.wp.com/latex.php?latex=%7BB%7D&bg=ffffff&fg=000000&s=0&c=20201002) as a lattice in a ![{D}](https://s0.wp.com/latex.php?latex=%7BD%7D&bg=ffffff&fg=000000&s=0&c=20201002)-dimensional vector space, which (due to the presence of ![{i}](https://s0.wp.com/latex.php?latex=%7Bi%7D&bg=ffffff&fg=000000&s=0&c=20201002), which excludes purely real embeddings) is naturally thought of as the product of ![{D/2}](https://s0.wp.com/latex.php?latex=%7BD%2F2%7D&bg=ffffff&fg=000000&s=0&c=20201002) copies of ![{{\bf C}}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cbf+C%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002). For instance, in the running example, one can identify a ring element ![{B}](https://s0.wp.com/latex.php?latex=%7BB%7D&bg=ffffff&fg=000000&s=0&c=20201002) with an element

![\displaystyle ( a + b i + c \sqrt{3} + d i \sqrt{3}, a + b i + c \sqrt{3} + d i \sqrt{3} )](https://s0.wp.com/latex.php?latex=%5Cdisplaystyle+%28+a+%2B+b+i+%2B+c+%5Csqrt%7B3%7D+%2B+d+i+%5Csqrt%7B3%7D%2C+a+%2B+b+i+%2B+c+%5Csqrt%7B3%7D+%2B+d+i+%5Csqrt%7B3%7D+%29&bg=ffffff&fg=000000&s=0&c=20201002) of ![{{\bf C}^2}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cbf+C%7D%5E2%7D&bg=ffffff&fg=000000&s=0&c=20201002), and with this embedding ![{B}](https://s0.wp.com/latex.php?latex=%7BB%7D&bg=ffffff&fg=000000&s=0&c=20201002) becomes a lattice in ![{{\bf C}^2}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cbf+C%7D%5E2%7D&bg=ffffff&fg=000000&s=0&c=20201002). In general, the embedding of ![{B}](https://s0.wp.com/latex.php?latex=%7BB%7D&bg=ffffff&fg=000000&s=0&c=20201002) into ![{{\bf C}^{D/2}}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cbf+C%7D%5E%7BD%2F2%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) is essentially the [Walsh–Fourier transform](https://en.wikipedia.org/wiki/Hadamard_transform), weighted by the various square roots of ![{q_1,\dots,q_g}](https://s0.wp.com/latex.php?latex=%7Bq_1%2C%5Cdots%2Cq_g%7D&bg=ffffff&fg=000000&s=0&c=20201002). (The appearance of the Walsh-Fourier transform reflects the fact that the specific number field we are working with is an abelian Galois extension with Galois group ![{({\mathbb Z}/2{\mathbb Z})^{g+1}}](https://s0.wp.com/latex.php?latex=%7B%28%7B%5Cmathbb+Z%7D%2F2%7B%5Cmathbb+Z%7D%29%5E%7Bg%2B1%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002).) Because of this, one can readily compute the covolume of the lattice, which up to lower order terms is basically ![{(\sqrt{q_1} \dots \sqrt{q_g})^D}](https://s0.wp.com/latex.php?latex=%7B%28%5Csqrt%7Bq_1%7D+%5Cdots+%5Csqrt%7Bq_g%7D%29%5ED%7D&bg=ffffff&fg=000000&s=0&c=20201002), which with our construction can be crudely bounded by ![{G^{O(gD)}}](https://s0.wp.com/latex.php?latex=%7BG%5E%7BO%28gD%29%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002). As long as we keep ![{g}](https://s0.wp.com/latex.php?latex=%7Bg%7D&bg=ffffff&fg=000000&s=0&c=20201002) small compared to ![{t/\log t}](https://s0.wp.com/latex.php?latex=%7Bt%2F%5Clog+t%7D&bg=ffffff&fg=000000&s=0&c=20201002), this covolume will be small compared to ![{2^{tD}}](https://s0.wp.com/latex.php?latex=%7B2%5E%7BtD%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) and will end up being a lower order term.

The reason we care about the covolume (whichis essentially the square root of the [discriminant](https://en.wikipedia.org/wiki/Discriminant_of_an_algebraic_number_field)) is because of Minkowski’s theorem, which we will use in this crude form: [Minkowski’s theorem](https://en.wikipedia.org/wiki/Minkowski%27s_theorem): any lattice ![{\Lambda}](https://s0.wp.com/latex.php?latex=%7B%5CLambda%7D&bg=ffffff&fg=000000&s=0&c=20201002) in a ![{D}](https://s0.wp.com/latex.php?latex=%7BD%7D&bg=ffffff&fg=000000&s=0&c=20201002)-dimensional space of covolume ![{V}](https://s0.wp.com/latex.php?latex=%7BV%7D&bg=ffffff&fg=000000&s=0&c=20201002) will contain a non-zero lattice vector of length ![{O(V^{1/D})}](https://s0.wp.com/latex.php?latex=%7BO%28V%5E%7B1%2FD%7D%29%7D&bg=ffffff&fg=000000&s=0&c=20201002). Thus for instance ![{B}](https://s0.wp.com/latex.php?latex=%7BB%7D&bg=ffffff&fg=000000&s=0&c=20201002) will contain some vector of length ![{O(Q^{g/2})}](https://s0.wp.com/latex.php?latex=%7BO%28Q%5E%7Bg%2F2%7D%29%7D&bg=ffffff&fg=000000&s=0&c=20201002), and any sublattice of ![{B}](https://s0.wp.com/latex.php?latex=%7BB%7D&bg=ffffff&fg=000000&s=0&c=20201002) of index ![{I}](https://s0.wp.com/latex.php?latex=%7BI%7D&bg=ffffff&fg=000000&s=0&c=20201002) will contain a vector of length ![{O(I^{1/D} G^{O(g)})}](https://s0.wp.com/latex.php?latex=%7BO%28I%5E%7B1%2FD%7D+G%5E%7BO%28g%29%7D%29%7D&bg=ffffff&fg=000000&s=0&c=20201002), and thus also of height ![{O(I^{1/D} G^{O(g)})}](https://s0.wp.com/latex.php?latex=%7BO%28I%5E%7B1%2FD%7D+G%5E%7BO%28g%29%7D%29%7D&bg=ffffff&fg=000000&s=0&c=20201002) by inverting the Walsh-Fourier transform.

Anyway, suppose that we can find some sublattice ![{\Lambda}](https://s0.wp.com/latex.php?latex=%7B%5CLambda%7D&bg=ffffff&fg=000000&s=0&c=20201002) (in fact, they will be ideals, but we will not need this) of ![{B}](https://s0.wp.com/latex.php?latex=%7BB%7D&bg=ffffff&fg=000000&s=0&c=20201002) for which we can obtain the inclusion

![\displaystyle \Lambda \cdot \overline{\Lambda} \subset m B, \ \ \ \ \ (9)](https://s0.wp.com/latex.php?latex=%5Cdisplaystyle+%5CLambda+%5Ccdot+%5Coverline%7B%5CLambda%7D+%5Csubset+m+B%2C+%5C+%5C+%5C+%5C+%5C+%289%29&bg=ffffff&fg=000000&s=0&c=20201002)

that is to say one has ![{w_1 \overline{w_2} = 0 \hbox{ mod } mB}](https://s0.wp.com/latex.php?latex=%7Bw_1+%5Coverline%7Bw_2%7D+%3D+0+%5Chbox%7B+mod+%7D+mB%7D&bg=ffffff&fg=000000&s=0&c=20201002) for all ![{w_1, w_2 \in \Lambda}](https://s0.wp.com/latex.php?latex=%7Bw_1%2C+w_2+%5Cin+%5CLambda%7D&bg=ffffff&fg=000000&s=0&c=20201002). Then clearly any element ![{w}](https://s0.wp.com/latex.php?latex=%7Bw%7D&bg=ffffff&fg=000000&s=0&c=20201002) of ![{\Lambda}](https://s0.wp.com/latex.php?latex=%7B%5CLambda%7D&bg=ffffff&fg=000000&s=0&c=20201002) will obey the congruence [(7)](#w-displace-cong). An obvious choice of ![{\Lambda}](https://s0.wp.com/latex.php?latex=%7B%5CLambda%7D&bg=ffffff&fg=000000&s=0&c=20201002) would be ![{\Lambda = mB}](https://s0.wp.com/latex.php?latex=%7B%5CLambda+%3D+mB%7D&bg=ffffff&fg=000000&s=0&c=20201002), but this is way too sparse: this lattice has index ![{m^D}](https://s0.wp.com/latex.php?latex=%7Bm%5ED%7D&bg=ffffff&fg=000000&s=0&c=20201002) in ![{B}](https://s0.wp.com/latex.php?latex=%7BB%7D&bg=ffffff&fg=000000&s=0&c=20201002), so the shortest vector one can hope to locate in it will have height ![{O(m G^{O(g)})}](https://s0.wp.com/latex.php?latex=%7BO%28m+G%5E%7BO%28g%29%7D%29%7D&bg=ffffff&fg=000000&s=0&c=20201002), which is too large for our purposes. Instead, we would like ![{\Lambda}](https://s0.wp.com/latex.php?latex=%7B%5CLambda%7D&bg=ffffff&fg=000000&s=0&c=20201002) to have index ![{m^{D/2}}](https://s0.wp.com/latex.php?latex=%7Bm%5E%7BD%2F2%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) (which is the smallest it can be while still yielding the inclusion [(9)](#lam)); then ![{\Lambda}](https://s0.wp.com/latex.php?latex=%7B%5CLambda%7D&bg=ffffff&fg=000000&s=0&c=20201002) will contain a non-zero element ![{w}](https://s0.wp.com/latex.php?latex=%7Bw%7D&bg=ffffff&fg=000000&s=0&c=20201002) of height ![{O(m^{1/2} G^{O(g)})}](https://s0.wp.com/latex.php?latex=%7BO%28m%5E%7B1%2F2%7D+G%5E%7BO%28g%29%7D%29%7D&bg=ffffff&fg=000000&s=0&c=20201002), which means that ![{w \overline{w}}](https://s0.wp.com/latex.php?latex=%7Bw+%5Coverline%7Bw%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) is equal to ![{m}](https://s0.wp.com/latex.php?latex=%7Bm%7D&bg=ffffff&fg=000000&s=0&c=20201002) times an element of ![{B}](https://s0.wp.com/latex.php?latex=%7BB%7D&bg=ffffff&fg=000000&s=0&c=20201002) of height ![{O(G^{O(g)})}](https://s0.wp.com/latex.php?latex=%7BO%28G%5E%7BO%28g%29%7D%29%7D&bg=ffffff&fg=000000&s=0&c=20201002). The number of such elements is ![{O(G^{O(gD)})}](https://s0.wp.com/latex.php?latex=%7BO%28G%5E%7BO%28gD%29%7D%29%7D&bg=ffffff&fg=000000&s=0&c=20201002), which will end up being a lower order term that we can easily pigeonhole away.

We claim that we can find at least ![{2^{tD/4}}](https://s0.wp.com/latex.php?latex=%7B2%5E%7BtD%2F4%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) different sublattices ![{\Lambda}](https://s0.wp.com/latex.php?latex=%7B%5CLambda%7D&bg=ffffff&fg=000000&s=0&c=20201002) of index ![{m^{D/2}}](https://s0.wp.com/latex.php?latex=%7Bm%5E%7BD%2F2%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) that obey [(9)](#lam). To verify this claim, it is a straightforward matter to use the Chinese remainder theorem to work “prime by prime”. Indeed, it suffices to show for each of the ![{t}](https://s0.wp.com/latex.php?latex=%7Bt%7D&bg=ffffff&fg=000000&s=0&c=20201002) primes ![{p}](https://s0.wp.com/latex.php?latex=%7Bp%7D&bg=ffffff&fg=000000&s=0&c=20201002) dividing ![{m}](https://s0.wp.com/latex.php?latex=%7Bm%7D&bg=ffffff&fg=000000&s=0&c=20201002), that there are ![{2^{D/4}}](https://s0.wp.com/latex.php?latex=%7B2%5E%7BD%2F4%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) different sublattices ![{\Lambda_p}](https://s0.wp.com/latex.php?latex=%7B%5CLambda_p%7D&bg=ffffff&fg=000000&s=0&c=20201002) of ![{B}](https://s0.wp.com/latex.php?latex=%7BB%7D&bg=ffffff&fg=000000&s=0&c=20201002) of index ![{p^{D/2}}](https://s0.wp.com/latex.php?latex=%7Bp%5E%7BD%2F2%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) that obey the inclusion

![\displaystyle \Lambda_p \cdot \overline{\Lambda_p} \subset p B. ](https://s0.wp.com/latex.php?latex=%5Cdisplaystyle+%5CLambda_p+%5Ccdot+%5Coverline%7B%5CLambda_p%7D+%5Csubset+p+B.+&bg=ffffff&fg=000000&s=0&c=20201002) We can descend now to the finite ring ![{B/pB}](https://s0.wp.com/latex.php?latex=%7BB%2FpB%7D&bg=ffffff&fg=000000&s=0&c=20201002), which is a ![{D}](https://s0.wp.com/latex.php?latex=%7BD%7D&bg=ffffff&fg=000000&s=0&c=20201002)-dimensional vector space over the finite field ![{{\mathbb F}_p}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cmathbb+F%7D_p%7D&bg=ffffff&fg=000000&s=0&c=20201002). The complex conjugation operation descends to an involution on this vector space, and we are looking for ![{2^{D/4}}](https://s0.wp.com/latex.php?latex=%7B2%5E%7BD%2F4%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) subspaces ![{V}](https://s0.wp.com/latex.php?latex=%7BV%7D&bg=ffffff&fg=000000&s=0&c=20201002) of dimension ![{D/2}](https://s0.wp.com/latex.php?latex=%7BD%2F2%7D&bg=ffffff&fg=000000&s=0&c=20201002) with the property that

![\displaystyle V \cdot \overline{V} = 0.](https://s0.wp.com/latex.php?latex=%5Cdisplaystyle++V+%5Ccdot+%5Coverline%7BV%7D+%3D+0.&bg=ffffff&fg=000000&s=0&c=20201002)

So now we just need to understand the structure of ![{B/pB}](https://s0.wp.com/latex.php?latex=%7BB%2FpB%7D&bg=ffffff&fg=000000&s=0&c=20201002). The general [Wedderburn–Artin theorem](https://en.wikipedia.org/wiki/Wedderburn%E2%80%93Artin_theorem) tells us that this ring is the product of finite fields, but we can be much more explicit here in this specific situation. Exactly as Minkowski embedding maps rings in number fields into product of copies of ![{{\bf R}}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cbf+R%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) and ![{{\bf C}}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cbf+C%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) associated to the real and complex embeddings of the ring, we can also embed ![{B/pB}](https://s0.wp.com/latex.php?latex=%7BB%2FpB%7D&bg=ffffff&fg=000000&s=0&c=20201002) into a product of copies of ![{{\mathbb F}_p}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cmathbb+F%7D_p%7D&bg=ffffff&fg=000000&s=0&c=20201002) and ![{{\mathbb F}_{p^2}}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cmathbb+F%7D_%7Bp%5E2%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002), depending on how we assign square roots to ![{-1}](https://s0.wp.com/latex.php?latex=%7B-1%7D&bg=ffffff&fg=000000&s=0&c=20201002) or ![{q_1,\dots,q_g}](https://s0.wp.com/latex.php?latex=%7Bq_1%2C%5Cdots%2Cq_g%7D&bg=ffffff&fg=000000&s=0&c=20201002) in ![{{\mathbb F}_p}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cmathbb+F%7D_p%7D&bg=ffffff&fg=000000&s=0&c=20201002) or the quadratic extension ![{{\mathbb F}_{p^2}}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cmathbb+F%7D_%7Bp%5E2%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002). We can illustrate this with the running example. As ![{p = -1}](https://s0.wp.com/latex.php?latex=%7Bp+%3D+-1%7D&bg=ffffff&fg=000000&s=0&c=20201002) mod ![{4}](https://s0.wp.com/latex.php?latex=%7B4%7D&bg=ffffff&fg=000000&s=0&c=20201002), the square roots ![{\alpha, -\alpha}](https://s0.wp.com/latex.php?latex=%7B%5Calpha%2C+-%5Calpha%7D&bg=ffffff&fg=000000&s=0&c=20201002) of ![{-1}](https://s0.wp.com/latex.php?latex=%7B-1%7D&bg=ffffff&fg=000000&s=0&c=20201002) in ![{{\mathbb F}_p}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cmathbb+F%7D_p%7D&bg=ffffff&fg=000000&s=0&c=20201002) stay in ![{{\mathbb F}_p}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cmathbb+F%7D_p%7D&bg=ffffff&fg=000000&s=0&c=20201002). Suppose first that ![{3}](https://s0.wp.com/latex.php?latex=%7B3%7D&bg=ffffff&fg=000000&s=0&c=20201002) also splits into square roots ![{\beta, -\beta}](https://s0.wp.com/latex.php?latex=%7B%5Cbeta%2C+-%5Cbeta%7D&bg=ffffff&fg=000000&s=0&c=20201002) in ![{{\mathbb F}_p}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cmathbb+F%7D_p%7D&bg=ffffff&fg=000000&s=0&c=20201002). Then we can embed ![{B/pB}](https://s0.wp.com/latex.php?latex=%7BB%2FpB%7D&bg=ffffff&fg=000000&s=0&c=20201002) into ![{{\mathbb F}_p^4}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cmathbb+F%7D_p%5E4%7D&bg=ffffff&fg=000000&s=0&c=20201002) by mapping

![\displaystyle a + b i + c \sqrt{3} + d i \sqrt{3} \mapsto (a + b \alpha + c \beta + d \alpha \beta, a + b \alpha + c \beta - d \alpha \beta, a - b \alpha + c \beta + d \alpha \beta, a - b \alpha + c \beta - d \alpha \beta).](https://s0.wp.com/latex.php?latex=%5Cdisplaystyle++a+%2B+b+i+%2B+c+%5Csqrt%7B3%7D+%2B+d+i+%5Csqrt%7B3%7D+%5Cmapsto+%28a+%2B+b+%5Calpha+%2B+c+%5Cbeta+%2B+d+%5Calpha+%5Cbeta%2C+a+%2B+b+%5Calpha+%2B+c+%5Cbeta+-+d+%5Calpha+%5Cbeta%2C+a+-+b+%5Calpha+%2B+c+%5Cbeta+%2B+d+%5Calpha+%5Cbeta%2C+a+-+b+%5Calpha+%2B+c+%5Cbeta+-+d+%5Calpha+%5Cbeta%29.&bg=ffffff&fg=000000&s=0&c=20201002) By counting elements we see that this embedding is in fact an isomorphism. If instead ![{3}](https://s0.wp.com/latex.php?latex=%7B3%7D&bg=ffffff&fg=000000&s=0&c=20201002) does not split, so that ![{\pm \beta}](https://s0.wp.com/latex.php?latex=%7B%5Cpm+%5Cbeta%7D&bg=ffffff&fg=000000&s=0&c=20201002) now lie in ![{{\mathbb F}_{p^2}}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cmathbb+F%7D_%7Bp%5E2%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002), then we can embed ![{B/pB}](https://s0.wp.com/latex.php?latex=%7BB%2FpB%7D&bg=ffffff&fg=000000&s=0&c=20201002) into ![{ {\mathbb F}_{p^2}^2}](https://s0.wp.com/latex.php?latex=%7B+%7B%5Cmathbb+F%7D_%7Bp%5E2%7D%5E2%7D&bg=ffffff&fg=000000&s=0&c=20201002) by mapping

![\displaystyle a + b i + c \sqrt{3} + d i \sqrt{3} \mapsto (a + b \alpha + c \beta + d \alpha \beta, a - b \alpha + c \beta + d \alpha \beta),](https://s0.wp.com/latex.php?latex=%5Cdisplaystyle++a+%2B+b+i+%2B+c+%5Csqrt%7B3%7D+%2B+d+i+%5Csqrt%7B3%7D+%5Cmapsto+%28a+%2B+b+%5Calpha+%2B+c+%5Cbeta+%2B+d+%5Calpha+%5Cbeta%2C+a+-+b+%5Calpha+%2B+c+%5Cbeta+%2B+d+%5Calpha+%5Cbeta%29%2C&bg=ffffff&fg=000000&s=0&c=20201002) thus we drop half of the previous embeddings as being conjugate to the half that we retain. Again, counting shows that this is an isomorphism.

In general, one can show that ![{B/pB}](https://s0.wp.com/latex.php?latex=%7BB%2FpB%7D&bg=ffffff&fg=000000&s=0&c=20201002) is isomorphic to either ![{{\mathbb F}_p^D}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cmathbb+F%7D_p%5ED%7D&bg=ffffff&fg=000000&s=0&c=20201002) (if all the ![{q_1,\dots,q_g}](https://s0.wp.com/latex.php?latex=%7Bq_1%2C%5Cdots%2Cq_g%7D&bg=ffffff&fg=000000&s=0&c=20201002) split in ![{{\mathbb F}_p}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cmathbb+F%7D_p%7D&bg=ffffff&fg=000000&s=0&c=20201002)) or ![{{\mathbb F}_{p^2}^{D/2}}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cmathbb+F%7D_%7Bp%5E2%7D%5E%7BD%2F2%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) (if at least one of the ![{q_1,\dots,q_g}](https://s0.wp.com/latex.php?latex=%7Bq_1%2C%5Cdots%2Cq_g%7D&bg=ffffff&fg=000000&s=0&c=20201002) does not split). Furthermore, in the former case the ![{D}](https://s0.wp.com/latex.php?latex=%7BD%7D&bg=ffffff&fg=000000&s=0&c=20201002) copies of ![{{\mathbb F}_p}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cmathbb+F%7D_p%7D&bg=ffffff&fg=000000&s=0&c=20201002) organize into ![{D/2}](https://s0.wp.com/latex.php?latex=%7BD%2F2%7D&bg=ffffff&fg=000000&s=0&c=20201002) conjugate pairs (corresponding to flipping ![{\alpha}](https://s0.wp.com/latex.php?latex=%7B%5Calpha%7D&bg=ffffff&fg=000000&s=0&c=20201002) to ![{-\alpha}](https://s0.wp.com/latex.php?latex=%7B-%5Calpha%7D&bg=ffffff&fg=000000&s=0&c=20201002)), and in the latter case the ![{D/2}](https://s0.wp.com/latex.php?latex=%7BD%2F2%7D&bg=ffffff&fg=000000&s=0&c=20201002) copies of ![{{\mathbb F}_{p^2}}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cmathbb+F%7D_%7Bp%5E2%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) organize into ![{D/4}](https://s0.wp.com/latex.php?latex=%7BD%2F4%7D&bg=ffffff&fg=000000&s=0&c=20201002) conjugate pairs. By selecting one element from each conjugate pair, and taking ![{V}](https://s0.wp.com/latex.php?latex=%7BV%7D&bg=ffffff&fg=000000&s=0&c=20201002) to be the joint kernel of such elements, we can generate either ![{2^{D/2}}](https://s0.wp.com/latex.php?latex=%7B2%5E%7BD%2F2%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) or ![{2^{D/4}}](https://s0.wp.com/latex.php?latex=%7B2%5E%7BD%2F4%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) different subspaces ![{V}](https://s0.wp.com/latex.php?latex=%7BV%7D&bg=ffffff&fg=000000&s=0&c=20201002) of dimension ![{D/2}](https://s0.wp.com/latex.php?latex=%7BD%2F2%7D&bg=ffffff&fg=000000&s=0&c=20201002) with the desired property that ![{V \cdot \overline{V} = 0}](https://s0.wp.com/latex.php?latex=%7BV+%5Ccdot+%5Coverline%7BV%7D+%3D+0%7D&bg=ffffff&fg=000000&s=0&c=20201002). By the aforementioned Chinese remainder argument, this gives the ![{2^{tD/4}}](https://s0.wp.com/latex.php?latex=%7B2%5E%7BtD%2F4%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) claimed lattices ![{\Lambda}](https://s0.wp.com/latex.php?latex=%7B%5CLambda%7D&bg=ffffff&fg=000000&s=0&c=20201002) of index ![{m^{D/2}}](https://s0.wp.com/latex.php?latex=%7Bm%5E%7BD%2F2%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002).

Invoking Minkowski’s theorem, this now generates ![{2^{tD/4}}](https://s0.wp.com/latex.php?latex=%7B2%5E%7BtD%2F4%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) non-zero vectors ![{w}](https://s0.wp.com/latex.php?latex=%7Bw%7D&bg=ffffff&fg=000000&s=0&c=20201002) of height ![{O( m^{1/2} G^{O(g)})}](https://s0.wp.com/latex.php?latex=%7BO%28+m%5E%7B1%2F2%7D+G%5E%7BO%28g%29%7D%29%7D&bg=ffffff&fg=000000&s=0&c=20201002) obeying [(7)](#w-displace-cong). There is a technical issue that some of these vectors could conceivably collide with each other; however there is a further Chinese remainder theorem argument (which I will omit here) that shows that any such ![{w}](https://s0.wp.com/latex.php?latex=%7Bw%7D&bg=ffffff&fg=000000&s=0&c=20201002) can belong to at most ![{O(G^{O(gD)})}](https://s0.wp.com/latex.php?latex=%7BO%28G%5E%7BO%28gD%29%7D%29%7D&bg=ffffff&fg=000000&s=0&c=20201002) such lattices.
